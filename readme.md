# python interactive learning platform project

## prerequisties : 
<ul>
<li>Nodejs</li>
<li>Mysql</li>
</ul>

## Database setup
<code> CREATE DATABASE  IF NOT EXISTS `nestdb`;
USE `nestdb`; </code>

## Server Instalaltion 

```bash
$ npm install
```
``` server is running on 3000
```
## Running the Server

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```



## Front application instalaltion 

```bash
$ npm install
```
``` front app is running on 8181
```
## Running for development
```
npm run serve
```

## Running for production
```
npm run build
```
